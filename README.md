## ğŸ® Usage Guide

1. **Open on Android device**
   - Navigate to the hosted URL in your Android browser

2. **Select Intensity Level**
   - Choose from Gentle, Medium, Intense, or Extreme
   - Use the Fine Tune slider for precise control

3. **Choose Vibration Pattern**
   - Select from 12 unique preset patterns
   - Each pattern has distinct characteristics

4. **Start Vibration**
   - Tap "Start Vibration" to begin
   - The button will show "Stop Vibration" when active

5. **Set Auto-Off Timer** (Optional)
   - Choose from Off, 10s, 30s, 1m, 2m, or 3m
   - Leave at "Off" for continuous operation

6. **Stop Vibration**
   - Tap "Stop Vibration" or wait for auto-off timer
